Bem-vindos ao Jogo Lab Run

|Como é o nosso jogo?

O jogo Lab Run, é um jogo inspirado no Jetpack Joyride, um fugitivo de um laboratório que possui um Jetpack de metralhadora e seu principal objetivo é desviar dos obstáculos até onde conseguir, e ai? vai encarar?

|Como executar o jogo?

  1. Baixe o arquivo no formato zip no botão "clique aqui para jogar"
  2. Descompacte o arquivo baixado
  3. Dentro da pasta do arquivo, crie um ambiente virtual python
  4. Instale Os requirements.txt dentro do ambiente
  5. Execute o arquivo "jogo.py"
